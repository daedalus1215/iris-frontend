# Iris

## Intro

- This is an AppleTV Remote App.
- This is the web version of the front end for iris. It ties in with iris-backend.

## Requirement

- copy `.env.sample` file and create a `.env` out of it.

## How to run

- node version 20+
- `npm install`
- `npm run dev`
